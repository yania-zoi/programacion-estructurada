# Participación 03: Batalla Naval
## Descripción
Juego de Batalla Naval para dos jugadores en consola. Cada jugador coloca 3 barcos en un tablero de 5x5 y por turnos dispara a coordenadas enemigas. El programa muestra el tablero enemigo, valida disparos repetidos o fuera de rango, e indica impactos, fallos y al ganador cuando se hunden todos los barcos rivales.
## Temas aplicados
- arreglos bidimensionales (tableros)
- funciones y paso de arreglos
- ciclos anidados para inicialización y visualización
- condicionales para validación de disparos y turnos
- entrada/salida con scanf, printf y fgets
- manejo de cadenas (nombres de jugadores)
## Compilación
gcc batallanaval.c -o batallanaval
## Ejecución
./batallanaval
