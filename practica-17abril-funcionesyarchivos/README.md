# Práctica 17 de abril: Funciones y Archivos
## Descripción
Programa que captura 6 números enteros, filtra los que son impares, los muestra en pantalla y guarda el resultado en un archivo de texto "resultado.txt". Utiliza funciones para modularizar la captura, visualización, filtrado y escritura en archivo.
## Temas aplicados
- funciones con paso de arreglos y punteros
- arreglos unidimensionales
- manejo de archivos (fopen, fprintf, fclose)
- filtrado de datos (números impares)
- entrada y salida estándar

gcc codigo-de-clase.c -o codigo-de-clase
## Ejecución
./codigo-de-clase
