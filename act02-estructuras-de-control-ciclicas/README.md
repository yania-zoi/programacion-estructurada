# Actividad 2: Estructuras de Control Cíclicas
## Descripción
Conjunto de tres programas en C que demuestran el uso de ciclos do-while y while. Incluyen un menú repetitivo hasta que el usuario decida salir, un sistema de promedio de calificaciones con validación de entrada, y una suma de números ingresados hasta que se introduzca un cero.
## Temas aplicados
- estructuras de control cíclicas (do-while, while, for)
- validación de entrada y filtrado de datos inválidos
- acumuladores y promedios
- condicionales dentro de ciclos
- entrada/salida estándar y limpieza de pantalla (system("cls"))
## Compilación
gcc dowhile.c -o dowhile
gcc promedio-ciclos.c -o promedio-ciclos
gcc whiledowhile.c -o whiledowhile
## Ejecución
./dowhile
./promedio-ciclos
./whiledowhile
