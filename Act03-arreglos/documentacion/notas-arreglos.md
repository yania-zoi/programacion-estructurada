Se toma a Float temp \[23\]; como un ejemplo de un arreglo, donde el temp es una dirección de memoria o un puntero, el 23 es la cantidad de datos que se van a almacenar en el arreglo. En la primera actividad se realiza la impresión de las variables temp1 y temp2, donde se imprime su dirección de memoria.

Para acceder al primer dato se utiliza printf(“%f”, temp\[0\]);, donde el primer valor se almacena en 0

Declaramos el valor del dato 1 y 2:

Temp\[0\] \= 20.5:  
Temp\[1\] \= 10.2:

Para imprimir los datos en la posicion 1 y 2 se ejecuta el codigo:

printf(“temp 1: %f\\n”, temp \[0\];  
printf(“temp 2: %f\\n”, temp \[1\];
