# Actividad 3: Arreglos
## Descripción
Programas que trabajan con arreglos unidimensionales de tipo float. El primero asigna valores directamente, muestra direcciones de memoria contiguas y accede a elementos por índice. El segundo captura temperaturas usando un ciclo for y luego las imprime, mostrando el uso de arreglos para almacenar múltiples valores.
## Temas aplicados
- arreglos unidimensionales
- índices y acceso a elementos
- direcciones de memoria y almacenamiento contiguo
- ciclos for para recorrido y llenado
- entrada y salida estándar (printf, scanf)
## Compilación
gcc arreglos01.c -o arreglos01
gcc arreglos01for.c -o arreglos01for
## Ejecución
./arreglos01
./arreglos01for
