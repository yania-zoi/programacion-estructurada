# Notas: Funciones
## Descripción
Tres programas que repasan conceptos básicos de funciones en C: retorno de valores, paso de parámetros por valor y manejo de arreglos bidimensionales como argumentos. Se muestra que modificar un parámetro dentro de la función no afecta la variable original cuando se pasa por valor, y cómo iterar e imprimir una matriz desde una función.
## Temas aplicados
- definición y llamado de funciones
- retorno de valores (return)
- paso de parámetros por valor
- arreglos bidimensionales como parámetros
- ciclos anidados para recorrer matrices
## Compilación
gcc funciones1.c -o funciones1
gcc funciones2.c -o funciones2
gcc funciones3.c -o funciones3
## Ejecución
./funciones1
./funciones2
./funciones3
