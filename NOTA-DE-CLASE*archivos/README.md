# Notas: Archivos
## Descripción
Programa que demuestra la escritura y lectura básica de archivos en C. Crea un archivo "tablero.txt", guarda el nombre del usuario y un tablero de 5x5 con caracteres representando el estado de una batalla naval, y posteriormente intenta leer y mostrar el nombre almacenado.
## Temas aplicados
- manejo de archivos (fopen, fprintf, fscanf, fclose)
- modos de apertura ("w")
- arreglos bidimensionales para representar un tablero
- entrada y salida estándar
- asignación de caracteres a celdas del tablero
## Compilación
gcc archivos.c -o archivos
## Ejecución
./archivos
