# Participación 13 de febrero: Condicionales
## Descripción
Conjunto de programas en C que utilizan estructuras condicionales (if, else, else if) para evaluar mayoría de edad, paridad de números, elegibilidad para beca según promedio y materias reprobadas, y validación de reinscripción según reglas académicas.
## Temas aplicados
- estructuras condicionales (if, else, else if)
- operadores relacionales y módulo (%)
- anidamiento de condiciones
- validación de entrada con múltiples criterios
- uso de scanf y printf para interacción con el usuario
## Compilación
gcc edades.c -o edades
gcc numero-impar.c -o numero-impar
gcc numero-par.c -o numero-par
gcc promedio.c -o promedio
gcc sistema-validacion.c -o sistema-validacion
## Ejecución
./edades
./numero-impar
./numero-par
./promedio
./sistema-validacion
