# Actividad 01-grupo01: Operadores

## Descripción
Actividad compuesta por dos programas en C que exploran el funcionamiento de diversos operadores y estructuras de control.
- **codigo 1** : Evalúa operadores de incremento/decremento ( `++`, `--` ), precedencia aritmética, división entera y formato de salida con `printf`.
- **codigo 2** : Corrige un fragmento de código que contiene errores de sintaxis (llaves, especificadores de formato) y de lógica en una estructura `if-else`, mostrando la versión corregida y funcional.

## Temas aplicados
- Operadores aritméticos y de incremento/decremento (`++`, `--`)
- Precedencia de operadores y uso de paréntesis
- División entera vs. división real (conversión implícita)
- Estructura selectiva `if` / `else`
- Buenas prácticas: indentación, llaves de bloque y orden lógico del código
- Función `printf` y especificadores de formato
