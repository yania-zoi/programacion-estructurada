# Práctica 20 de marzo: Strings
## Descripción
Programas que trabajan con cadenas de caracteres. El primero es una versión mejorada del juego Batalla Naval que utiliza arreglos de strings para los nombres de los jugadores. El segundo explora el almacenamiento de cadenas en arreglos bidimensionales, la captura con fgets y la impresión de direcciones de memoria de caracteres y enteros.
## Temas aplicados
- strings y arreglos de caracteres
- fgets para lectura segura de cadenas
- strcspn para eliminar saltos de línea
- arreglos bidimensionales de caracteres
- direcciones de memoria y su relación con tipos de datos
## Compilación
gcc batallanaval-RELOADED.c -o batallanaval-reloaded
gcc string.c -o string
## Ejecución
./batallanaval-reloaded
./string
