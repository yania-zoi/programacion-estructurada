# Actividad 1: Tipos de Datos
## Descripción
Programa que imprime el valor y la dirección de memoria de variables de distintos tipos de datos (char, int, float, double, long double, short, long, long long, unsigned) para calcular manualmente su tamaño en bytes. Se complementa con un ensayo sobre la historia de C.
## Temas aplicados
- tipos de datos y modificadores (short, long, unsigned)
- direcciones de memoria y operador &
- especificadores de formato (%c, %d, %f, %lf, %Lf, %hd, %ld, %lld, %u)
- cálculo del tamaño de tipos mediante aritmética de direcciones
## Compilación
gcc tipos.c -o tipos-de-datos
## Ejecución
./tipos-de-datos
